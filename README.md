## 📌 About The Project

A personal "About Me" page built as part of my Frontend Mentor practice. Designed to present personal information in a clean, readable layout with smooth interactivity.

---

## 🛠️ Built With

- Semantic HTML5
- CSS3 (Flexbox, Custom Properties)
- Vanilla JavaScript
- Responsive Design (Mobile-first)

---

## ✨ Features

- Clean personal profile layout
- Fully responsive across all screen sizes
- Smooth UI interactions with JavaScript
- Organized content sections

---

## 📚 What I Learned

- Structuring personal information with semantic HTML
- Building multi-section layouts with CSS Flexbox
- Adding interactivity with vanilla JavaScript
- Applying mobile-first responsive design principles
